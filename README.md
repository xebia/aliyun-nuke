# aliyun-nuke

Clears out all resources in any given Alibaba Cloud account. Use with extreme caution!

## Supports

Buckets{},
		ecs.Instances{},
		ecs.SecurityGroups{},
		vpc.Vpcs{},
		vpc.VSwitches{},
